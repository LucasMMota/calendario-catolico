# Calendario Catolico

Calendario liturgico catolico em formato ICS, cobrindo de 2025 a 2035.

Inclui solenidades, festas, memorias, tempos liturgicos (Quaresma, Advento, Pascoa) e devocao popular.

## Como assinar

Clique no link abaixo ou adicione manualmente no seu app de calendario:

```
webcal://raw.githubusercontent.com/LucasMMota/calendario-catolico/main/calendario_catolico.ics
```

No iPhone/iPad, basta tocar no link acima. No Google Calendar, adicione via "Outros calendarios > Por URL".
